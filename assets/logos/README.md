## Verwendung

Diese Logos werden in folgenden Projekten verwendet:

- `docker/frontend-website/` - Haupt-Website
- `docker/wiki-docs/` - Dokumentation
- `simple-order-system/frontend/` - Kassen-System
- `simple-inventory-system/frontend/` - Inventar-Verwaltung
- `vorstand-android/` - Android Vorstand-App
- `simple-order-system/kitchen-display-android/` - Android KDS App

Bei Änderungen sollten die Logos an allen relevanten Stellen aktualisiert werden.
