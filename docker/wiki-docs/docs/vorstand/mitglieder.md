# Mitgliederverwaltung

## Mitglied hinzufügen

1. Gehe zu "Mitglieder"
2. Klicke auf "+ Neues Mitglied"
3. Fülle die Pflichtfelder aus:
   - Vorname, Nachname
   - E-Mail-Adresse
   - Status (Aktiv/Passiv/Ehren)
4. Optional: Funktion, Foto, T-Shirt-Grösse
5. Klicke auf "Speichern"

:::tip
Nach dem Speichern wird automatisch ein Authentik-Account erstellt.
:::

## Mitglied bearbeiten

1. Klicke auf das Mitglied in der Liste
2. Ändere die gewünschten Felder
3. Klicke auf "Speichern"

## Funktion zuweisen

Über das Dropdown "Funktion" kannst du zuweisen:

| Funktion | Beschreibung |
|----------|--------------|
| Admin | Volle Rechte im System |
| Präsident | Vorstand |
| Aktuar | Vorstand |
| Kassier | Vorstand |
| Materialwart | Vorstand |
| Beisitzer | Vorstand |
| Revisor | Kein Vorstand |

## Mitglied löschen

1. Öffne das Mitglied
2. Klicke auf "Löschen"
3. Bestätige die Löschung

:::warning
Gelöschte Mitglieder können nicht wiederhergestellt werden!
:::

## Export

Die Mitgliederliste kann als CSV exportiert werden:
1. Gehe zu "Mitglieder"
2. Klicke auf "Export"
3. Wähle das Format (CSV/Excel)
