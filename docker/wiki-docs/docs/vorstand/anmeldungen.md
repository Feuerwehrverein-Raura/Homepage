# Anmeldungen verwalten

## Übersicht

Unter "Anmeldungen" siehst du alle eingegangenen Helfer-Anmeldungen.

## Anmeldung genehmigen

1. Klicke auf eine ausstehende Anmeldung
2. Prüfe die Details (Name, E-Mail, Schichten)
3. Klicke auf "Genehmigen"

Der Helfer erhält automatisch eine Bestätigungs-E-Mail.

## Anmeldung ablehnen

1. Öffne die Anmeldung
2. Optional: Gib einen Grund an
3. Klicke auf "Ablehnen"

Der Helfer wird per E-Mail informiert.

## Status-Übersicht

| Status | Bedeutung |
|--------|-----------|
| Ausstehend | Wartet auf Bearbeitung |
| Genehmigt | Helfer ist eingeteilt |
| Abgelehnt | Anmeldung wurde abgelehnt |

## Erinnerungen senden

Schicht-Erinnerungen werden automatisch am Tag vor dem Einsatz versendet.

Manuelle Erinnerung:
1. Gehe zu "Events" > [Event] > "Schichten"
2. Klicke auf "Erinnerungen senden"
