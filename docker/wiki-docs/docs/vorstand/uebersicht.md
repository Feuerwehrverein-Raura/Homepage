# Vorstand-Übersicht

Der Vorstand-Bereich ermöglicht die Verwaltung des Vereins.

## Zugang

URL: [fwv-raura.ch/vorstand.html](https://fwv-raura.ch/vorstand.html)

## Funktionen

| Bereich | Beschreibung |
|---------|--------------|
| **Mitglieder** | Mitglieder hinzufügen, bearbeiten, löschen |
| **Events** | Anlässe erstellen und verwalten |
| **Anmeldungen** | Helfer-Anmeldungen genehmigen/ablehnen |
| **E-Mail** | Mailcow-Postfächer verwalten |
| **Audit-Log** | Aktivitäten einsehen |

## Berechtigungen

| Rolle | Rechte |
|-------|--------|
| Vorstand | Lesen, Erstellen, Bearbeiten |
| Admin | Volle Rechte inkl. Löschen |

## Navigation

Nach dem Login siehst du die Hauptnavigation:

- **Dashboard** - Übersicht und Statistiken
- **Mitglieder** - Mitgliederverwaltung
- **Events** - Event-Management
- **E-Mail** - Mailcow-Integration
