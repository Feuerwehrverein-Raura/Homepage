# Event-Management

## Event erstellen

1. Gehe zu "Events"
2. Klicke auf "+ Neuer Anlass"
3. Fülle die Details aus:
   - Titel und Beschreibung
   - Datum und Zeit
   - Ort
   - Kategorie
4. Füge Schichten hinzu (optional)
5. Klicke auf "Speichern"

## Schichten verwalten

Schichten ermöglichen die Einteilung von Helfern:

1. Öffne ein Event
2. Gehe zu "Schichten"
3. Klicke auf "+ Neue Schicht"
4. Definiere:
   - Name (z.B. "Bar Morgen")
   - Datum und Zeit
   - Anzahl benötigter Helfer
   - Bereich

## Event-Status

| Status | Beschreibung |
|--------|--------------|
| Geplant | Event ist in Planung |
| Aktiv | Anmeldungen möglich |
| Abgeschlossen | Event ist vorbei |
| Abgesagt | Event wurde abgesagt |

## Kalender-Integration

Events werden automatisch als iCal-Feed bereitgestellt:

```
https://events.fwv-raura.ch/calendar/ics
```

Diesen Link kannst du in Google Calendar, Outlook etc. abonnieren.
