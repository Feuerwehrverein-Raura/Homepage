# Mailcow E-Mail-Verwaltung

Im Vorstand-Bereich kannst du E-Mail-Postfächer und Aliases verwalten.

## Postfächer anzeigen

1. Gehe zu "E-Mail"
2. Siehst du alle Postfächer für @fwv-raura.ch

| Postfach | Zweck |
|----------|-------|
| praesident@ | Präsident |
| aktuar@ | Aktuar |
| kassier@ | Kassier |
| materialwart@ | Materialwart |
| beisitzer@ | Beisitzer |
| info@ | Allgemeine Anfragen |
| social-media@ | Social Media (geteiltes Postfach) |

## Speicherplatz

Für jedes Postfach wird der belegte Speicherplatz angezeigt:
- **Verwendet**: Aktuelle Nutzung
- **Quota**: Maximale Grösse

## Aliases

Aliases leiten E-Mails an andere Adressen weiter.

### Beispiele
- `kontakt@fwv-raura.ch` → `info@fwv-raura.ch`
- `webmaster@fwv-raura.ch` → `aktuar@fwv-raura.ch`

## Passwort ändern
