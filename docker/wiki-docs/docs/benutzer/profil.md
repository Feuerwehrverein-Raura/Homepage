# Profil verwalten

Als Mitglied kannst du einige deiner persönlichen Daten selbst verwalten. Diese Anleitung zeigt dir, wie.

## Mein Bereich öffnen

1. Melde dich auf [fwv-raura.ch](https://fwv-raura.ch) an
2. Klicke oben rechts auf deinen Namen
3. Wähle **"Mein Bereich"** oder **"Profil"**

## Daten ansehen

Im Profil siehst du alle gespeicherten Daten:

### Persönliche Daten
| Feld | Bearbeitbar |
|------|-------------|
| Anrede | Nein |
| Vorname | Nein |
| Nachname | Nein |
| Geburtsdatum | Nein |

### Kontaktdaten
| Feld | Bearbeitbar |
|------|-------------|
| E-Mail (Login) | Nein* |
| Telefon | Ja |
| Mobile | Ja |
| Versand-E-Mail | Ja |

*Die Login-E-Mail kann nur vom Vorstand geändert werden.

### Adresse
| Feld | Bearbeitbar |
|------|-------------|
| Strasse | Ja |
| Adresszusatz | Ja |
| PLZ | Ja |
| Ort | Ja |

### Mitgliedschaft
| Feld | Bearbeitbar |
|------|-------------|
| Status | Nein |
| Eintrittsdatum | Nein |
| Funktion | Nein |

## Daten ändern

### Schritt-für-Schritt

1. Öffne deinen **Mein Bereich**
2. Klicke auf **"Profil bearbeiten"** oder das Stift-Symbol
3. Ändere die gewünschten Felder
4. Klicke auf **"Speichern"**
5. Du siehst eine Bestätigung

### Was kann ich ändern?

Du kannst selbst ändern:
- **Telefonnummern** - Festnetz und Mobile
- **Adresse** - Strasse, PLZ, Ort
- **Versand-E-Mail** - Falls anders als Login-E-Mail
- **IBAN** - Für Rückerstattungen
- **Bemerkungen** - Zusätzliche Infos

### Was muss der Vorstand ändern?

Folgende Daten kann nur der Vorstand ändern:
- Name (Vorname, Nachname)
- Geburtsdatum
- Login-E-Mail
- Mitgliedsstatus (Aktiv/Passiv/Ehren)
- Funktionen (Vorstand, Revisor, etc.)

Bei Änderungswünschen kontaktiere [aktuar@fwv-raura.ch](mailto:aktuar@fwv-raura.ch).

## Profilbild

### Bild hochladen

1. Öffne dein Profil
2. Klicke auf das Bild-Symbol oder **"Foto ändern"**
3. Wähle ein Bild von deinem Gerät
4. Das Bild wird automatisch hochgeladen

**Anforderungen:**
- Format: JPG oder PNG
- Maximale Grösse: 5 MB
- Empfohlen: Quadratisches Bild

### Bild löschen

1. Öffne dein Profil
2. Klicke auf das **X** beim Bild
3. Bestätige die Löschung

## Benachrichtigungen

Du kannst einstellen, welche Benachrichtigungen du erhalten möchtest.

### Verfügbare Einstellungen

| Benachrichtigung | Beschreibung |
|-----------------|--------------|
| **Schicht-Erinnerungen** | E-Mail 24h vor deinem Einsatz |
| **Event-Updates** | Bei Änderungen an Events, für die du angemeldet bist |
| **Newsletter** | Allgemeine Vereins-Neuigkeiten |
| **Rundschreiben** | Wichtige Mitteilungen vom Vorstand |

### Einstellungen ändern

1. Öffne deinen **Mein Bereich**
2. Gehe zu **"Benachrichtigungen"** oder **"Einstellungen"**
3. Aktiviere/deaktiviere die gewünschten Optionen
4. Speichere

:::info
Wichtige Vereinsmitteilungen erhältst du unabhängig von diesen Einstellungen.
:::

## Zustellpräferenzen

Du kannst wählen, wie du Vereinspost erhalten möchtest.

### E-Mail-Zustellung
- Rundschreiben per E-Mail
- Schnell und kostenlos
- Empfohlen

### Post-Zustellung
- Rundschreiben per Briefpost
- Für Mitglieder ohne E-Mail
- Längere Lieferzeit

### Beide aktivieren
Du kannst auch beide Optionen aktivieren, um nichts zu verpassen.

### Ändern

1. Öffne dein Profil
2. Finde die Sektion **"Zustellung"** oder **"Versand"**
3. Wähle deine Präferenz
4. Speichere

## Meine Anmeldungen

Im Mein Bereich siehst du auch:

### Aktive Anmeldungen
- Events, für die du angemeldet bist
- Deine zugewiesenen Schichten
- Status deiner Anmeldungen

### Vergangene Events
- Events, an denen du teilgenommen hast
- Deine geleisteten Schichten

## Meine Schichten

Übersicht deiner Helfereinsätze:

### Kommende Schichten
- Datum und Uhrzeit
- Event und Bereich (Bar, Küche, etc.)
- Andere Helfer in der gleichen Schicht

### Schicht absagen
Falls du eine zugewiesene Schicht nicht wahrnehmen kannst:
1. Kontaktiere den Organisator oder Vorstand
2. So früh wie möglich melden
3. Hilf bei der Suche nach Ersatz

:::warning
Schichten selbst stornieren ist nicht möglich. Melde dich bitte beim Vorstand.
:::

## Passwort ändern

Siehe [Login → Passwort ändern](./login#passwort-ändern)

## Account löschen

Falls du deine Mitgliedschaft beenden möchtest:

1. Kontaktiere [aktuar@fwv-raura.ch](mailto:aktuar@fwv-raura.ch)
2. Dein Austritt wird bearbeitet
3. Dein Account wird deaktiviert
4. Deine Daten werden gemäss Datenschutz behandelt

Mehr dazu unter [Mitgliedschaft → Austritt](./mitgliedschaft#austritt)

## Datenschutz

### Welche Daten werden gespeichert?
- Kontaktdaten (Name, Adresse, E-Mail, Telefon)
- Mitgliedschaftsdaten (Status, Eintrittsdatum)
- Event-Teilnahmen und Schichten

### Wer sieht meine Daten?
- **Vorstand:** Alle Daten (für Vereinsverwaltung)
- **Andere Mitglieder:** Nur Name (z.B. im Arbeitsplan)
- **Öffentlich:** Nichts

### Daten exportieren
Du kannst eine Kopie deiner Daten anfordern. Kontaktiere [admin@fwv-raura.ch](mailto:admin@fwv-raura.ch).

## Probleme

### Änderungen werden nicht gespeichert
- Prüfe deine Internetverbindung
- Fülle alle Pflichtfelder aus
- Versuche es erneut
- Kontaktiere den Admin bei wiederholten Problemen

### Daten sind falsch
- Selbst änderbare Daten: Im Profil bearbeiten
- Andere Daten: Vorstand kontaktieren
