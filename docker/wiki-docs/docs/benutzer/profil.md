# Profil verwalten

## Persönliche Daten

Als Mitglied kannst du folgende Daten selbst ändern:

- Telefonnummer
- Mobile
- E-Mail-Adresse
- Versand-E-Mail
- Adresse (Strasse, PLZ, Ort)
- IBAN
- Bemerkungen

## Daten ändern

1. Melde dich im Mitgliederbereich an
2. Gehe zu "Mein Profil"
3. Ändere die gewünschten Felder
4. Klicke auf "Speichern"

:::note
Einige Felder wie Name, Geburtsdatum und Mitgliedsstatus können nur vom Vorstand geändert werden.
:::

## Benachrichtigungen

Du kannst einstellen, welche Benachrichtigungen du erhalten möchtest:

- **Schicht-Erinnerungen** - Erinnerung vor deinen Einsätzen
- **Event-Updates** - Änderungen an Anlässen
- **Newsletter** - Vereins-Neuigkeiten
- **Allgemein** - Sonstige Mitteilungen
