# Login

## Mitglieder-Login (Authentik)

Mitglieder melden sich über Authentik an:

1. Klicke auf "Mitglieder-Login"
2. Du wirst zu auth.fwv-raura.ch weitergeleitet
3. Gib deine Zugangsdaten ein
4. Nach erfolgreicher Anmeldung wirst du zurückgeleitet

## Vorstand-Login (IMAP)

Vorstandsmitglieder haben zusätzlich Zugang zum Verwaltungsbereich:

1. Gehe zu [fwv-raura.ch/vorstand.html](https://fwv-raura.ch/vorstand.html)
2. Melde dich mit deiner @fwv-raura.ch E-Mail an
3. Das Passwort ist dein Mailcow-Passwort

## Admin-Login

Administratoren können sich mit dem Admin-Account anmelden:
- **E-Mail:** admin@fwv-raura.ch
- **Passwort:** Wird vom Server-Administrator verwaltet
