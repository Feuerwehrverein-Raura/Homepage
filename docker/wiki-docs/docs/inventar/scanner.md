# Scanner

Der integrierte Scanner ermöglicht schnelles Arbeiten mit Barcodes und QR-Codes.

## Funktionen

### Barcode scannen
1. Scanner-Symbol in der App klicken
2. Kamera auf Barcode richten
3. Artikel wird automatisch erkannt

### Aktionen nach Scan
- **Artikel gefunden**: Stock-Dialog öffnet sich
- **Artikel nicht gefunden**: Option zum Erstellen

## Stock-Dialog

Nach dem Scannen:

### Einbuchen (Stock In)
- Menge eingeben
- Optional: Notiz hinzufügen
- Bestand wird erhöht

### Ausbuchen (Stock Out)
- Menge eingeben
- Grund auswählen (Verbrauch, Defekt, etc.)
- Bestand wird reduziert

## Unterstützte Formate

- EAN-13 / EAN-8
- UPC-A / UPC-E
- Code 128 / Code 39
- QR-Codes
- Data Matrix

## QR-Codes

### In-App Scan
QR-Codes die in der App gescannt werden:
- URL wird erkannt
- Barcode wird extrahiert
- Normaler Stock-Dialog

### Externer Scan (Smartphone-Kamera)
QR-Codes die mit der Smartphone-Kamera gescannt werden:
- Öffnet öffentliche Artikel-Ansicht
- Zeigt Artikeldetails
- Kein Login erforderlich

## Tipps

### Gute Lichtverhältnisse
- Ausreichend Licht
- Keine Reflexionen
- Barcode nicht zu weit weg

### Kamera-Berechtigungen
- Browser fragt nach Kamera-Zugriff
- Einmal erlauben reicht
- In Browser-Einstellungen änderbar

### Offline-Nutzung
- Scanner funktioniert offline
- Änderungen werden gespeichert
- Sync bei Verbindung
