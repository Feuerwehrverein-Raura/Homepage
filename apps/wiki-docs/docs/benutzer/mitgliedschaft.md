# Mitgliedschaft

Alles rund um deine Mitgliedschaft beim Feuerwehrverein Raura.

## Mitglied werden

### Wer kann Mitglied werden?
Alle natürlichen Personen können Mitglied werden. Eine Zugehörigkeit zur aktiven Feuerwehr ist nicht erforderlich.

### Online-Anmeldung

1. Gehe zu [fwv-raura.ch](https://fwv-raura.ch)
2. Klicke auf **"Mitglied werden"**
3. Fülle das Formular aus:
   - Vorname und Nachname
   - Adresse
   - E-Mail-Adresse
   - Telefonnummer
   - Gewünschter Status (Aktiv/Passiv)
   - Feuerwehr-Zugehörigkeit (falls zutreffend)
   - Wie möchtest du Post erhalten (E-Mail/Brief)
4. Klicke auf **"Antrag absenden"**

### Was passiert danach?

1. **Antrag eingegangen**
   - Du erhältst eine Bestätigungs-E-Mail
   - Der Vorstand wird über deinen Antrag informiert

2. **Prüfung durch Vorstand**
   - Der Vorstand prüft deinen Antrag
   - Bei Fragen meldet sich jemand bei dir

3. **Entscheidung**
   - **Angenommen:** Du wirst im System erfasst und kannst dein Passwort setzen
   - **Abgelehnt:** Du erhältst eine Begründung per E-Mail

4. **Passwort setzen & Anmelden**
   - Gehe zu [auth.fwv-raura.ch](https://auth.fwv-raura.ch)
   - Klicke auf **"Passwort vergessen?"**
   - Gib deine E-Mail-Adresse ein
   - Du erhältst einen Reset-Link per E-Mail
   - Wähle ein sicheres Passwort
   - Danach kannst du dich auf der Website anmelden

### Aufnahmegebühr
Für neue Mitglieder fällt eine einmalige Aufnahmegebühr an. Der aktuelle Betrag wird dir im Antragsformular angezeigt.

## Mitglieder-Status

### Aktivmitglied

| | |
|-|-|
| **Beschreibung** | Vollmitglied mit allen Rechten und Pflichten |
| **Jahresbeitrag** | CHF 30.– (aktueller Stand) |
| **Rechte** | Stimmrecht an GV, Teilnahme an allen Anlässen |
| **Pflichten** | Teilnahme an Vereinsarbeit erwünscht |

### Passivmitglied

| | |
|-|-|
| **Beschreibung** | Unterstützendes Mitglied |
| **Jahresbeitrag** | CHF 20.– (aktueller Stand) |
| **Rechte** | Teilnahme an Anlässen (ohne Stimmrecht) |
| **Pflichten** | Keine aktive Mitarbeit erwartet |

### Ehrenmitglied

| | |
|-|-|
| **Beschreibung** | Vom Verein geehrte Person |
| **Jahresbeitrag** | Beitragsfrei |
| **Wie wird man Ehrenmitglied** | Ernennung durch GV für besondere Verdienste |

## Status wechseln

### Von Aktiv zu Passiv (oder umgekehrt)

1. Kontaktiere [aktuar@fwv-raura.ch](mailto:aktuar@fwv-raura.ch)
2. Gib deinen Wunsch an
3. Die Änderung wird zum nächsten Jahr wirksam
4. Du erhältst eine Bestätigung

### Fristen
- Statuswechsel bis 30. November für das Folgejahr
- Änderungen während des Jahres nach Absprache möglich

### Bezahlung

Du erhältst im Januar/Februar eine Rechnung:
- Per E-Mail (wenn E-Mail-Zustellung aktiviert)
- Per Post (wenn Post-Zustellung aktiviert)

### Zahlungsarten
- Einzahlung auf Vereinskonto (IBAN auf Rechnung)
- TWINT (falls angeboten)
- Bar an der GV

### Zahlungsfrist
30 Tage nach Rechnungsstellung. Bei Verzug erhältst du eine Mahnung.

### Rechnung nicht erhalten?
Kontaktiere [kassier@fwv-raura.ch](mailto:kassier@fwv-raura.ch).

## Vereinsarbeit

### Als Aktivmitglied
Von Aktivmitgliedern wird erwartet, dass sie sich bei Anlässen beteiligen:
- Helfen bei Dorffest, Fasnacht, etc.
- Teilnahme an Aufbau/Abbau
- Mithilfe bei Vereinsarbeit

### Als Passivmitglied
Passivmitglieder können, müssen aber nicht:
- Freiwillig bei Events helfen
- An Anlässen teilnehmen

### Helfer-Einsätze
Siehe [Events & Anmeldung](./events) für Details zur Anmeldung als Helfer.

## Vereinsvorteile

Als Mitglied profitierst du von:

### Veranstaltungen
- Einladung zu allen Vereinsanlässen
- Vergünstigte oder kostenlose Teilnahme
- Zugang zu Mitglieder-Events

### Gemeinschaft
- Teil einer aktiven Gemeinschaft
- Kontakte im Dorf
- Gesellige Anlässe

### Vergünstigungen
- Verpflegung bei Helfereinsätzen
- Helferbatzen bei grossen Events
- Vereinskleidung zu Vorzugspreisen

## Kontakt ändern

### Adressänderung
Wenn du umziehst:
1. Ändere deine Adresse im [Profil](./profil)
2. Oder melde die Änderung an [aktuar@fwv-raura.ch](mailto:aktuar@fwv-raura.ch)

### E-Mail ändern
Die Login-E-Mail kann nur vom Vorstand geändert werden:
1. Kontaktiere [aktuar@fwv-raura.ch](mailto:aktuar@fwv-raura.ch)
2. Gib die alte und neue E-Mail an
3. Die Änderung wird durchgeführt

### Telefonnummer ändern
Kannst du selbst im [Profil](./profil) ändern.

## Austritt

### Austritt beantragen

1. **E-Mail schreiben**
   - An [aktuar@fwv-raura.ch](mailto:aktuar@fwv-raura.ch)
   - Betreff: "Austritt aus dem FWV Raura"
   - Mit deinem Namen und dem gewünschten Austrittsdatum

2. **Frist beachten**
   - Austritt per Ende Jahr möglich
   - Kündigungsfrist: 30. November für das laufende Jahr

3. **Bestätigung erhalten**
   - Du erhältst eine Bestätigung des Austritts
   - Offene Beiträge werden noch fällig

### Was passiert nach dem Austritt?

- Dein Account wird deaktiviert
- Du hast keinen Zugang mehr zum Mitgliederbereich
- Du erhältst keine Vereinspost mehr
- Deine Daten werden nach gesetzlicher Frist gelöscht

### Wiedereintritt
Nach einem Austritt kannst du jederzeit wieder Mitglied werden. Es fällt keine erneute Aufnahmegebühr an.

## Häufige Fragen

### Muss ich bei der Feuerwehr sein?
Nein! Der FWV Raura ist ein eigenständiger Verein. Eine Feuerwehr-Zugehörigkeit ist nicht erforderlich.

### Was ist der Unterschied Aktiv/Passiv?
- **Aktiv:** Stimmrecht, voller Beitrag, Mitarbeit erwünscht
- **Passiv:** Kein Stimmrecht, reduzierter Beitrag, keine Mitarbeit erwartet

### Kann ich auch ohne Account mitmachen?
Für die meisten Anlässe brauchst du einen Account. Kontaktiere den Vorstand für Ausnahmen.

### Ich habe nie eine Willkommens-E-Mail erhalten
- Prüfe deinen Spam-Ordner
- Kontaktiere [admin@fwv-raura.ch](mailto:admin@fwv-raura.ch)

### Mein Antrag wurde abgelehnt, warum?
Kontaktiere [admin@fwv-raura.ch](mailto:admin@fwv-raura.ch) für eine Erklärung.
